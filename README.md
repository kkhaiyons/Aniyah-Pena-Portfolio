# Aniyah Pena Portfolio

This repository contains my personal portfolio website. It highlights my academic background, technical skills, and professional projects in a modern, responsive design. The site includes interactive elements such as a project carousel, contact form, and chatbot integration.

## Features

Responsive design that works across desktop, tablet, and mobile devices  
Navigation bar with quick access to Home, About, Projects, Resume, and Contact sections  
Social links to GitHub and LinkedIn  
About section describing my concentration in cybersecurity and professional experiences  
Skills grid showing programming, web development, and command line expertise  
Projects carousel with descriptions and GitHub links  
Resume section with options to download or view my resume directly from Google Drive  
Contact form integrated with Web3Forms API  
QR code footer for quick access to the repository  
Chatbot integration using Dialogflow Messenger  

## Navigation Guide

Home (#home)  
Introduction with profile picture, headline, and social links  

About (#about)  
Overview of academic concentration, professional experience, and skill cards  

Projects (#projects)  
Carousel showcasing featured projects  
Each project includes a description and GitHub link  
Dots below the carousel allow switching between projects  

Resume (#resume)  
Download Resume (direct file download)  
View Resume (opens in Google Drive viewer)  

Contact (#contact)  
Form with fields for name, email, and message  
Submit button sends the form via Web3Forms API  

Footer (.qr-footer)  
QR code that links to the portfolio repository  

## Technologies Used

HTML5 and CSS3 for structure and styling  
JavaScript for interactivity  
Figma for design prototyping  
Web3Forms API for contact form submissions  
Dialogflow Messenger for chatbot integration  

## Getting Started

Clone the repository:  
git clone https://github.com/kkhaiyons/Aniyah-Pena-Portfolio.git  

Open index.html in your browser  
Customize style.css and script.js for further styling or functionality
